This folder is served as static files by VitePress.

Place the *current* release artifacts here, for easy stable links:

- firmware/ots-fw-main.bin
- firmware/ots-fw-main-<tag>.bin
- userscript/userscript.ots.user.js
- hardware/kicad/*.zip
- hardware/fusion360/*.f3z or *.zip
- hardware/cad/*.step or *.zip

These files will be available at URLs like:
  /releases/latest/firmware/ots-fw-main.bin

Note: For large files or many releases, prefer attaching artifacts to GitHub Releases and linking to them from the docs.
